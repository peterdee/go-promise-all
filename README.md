## Emulating Promise.all() in Golang using sync.WaitGroup()

This example shows the usage of `sync.WaitGroup()` that helps with `Promise.all()` emulation.

### Deploy

```shell script
git clone https://github.com/peterdee/go-promise-all
cd ./go-promise-all
```

### Launch

```shell script
go run ./main.go
```
